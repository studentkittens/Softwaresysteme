Introduction

Flask is a microwebframework

  • keeps the core simple but extensible
    • no form validation
    • has no database abstraction layer
    • [ ... ]
  • add features by using extension

The idea of Flask is to build a good foundation for all applications. Everything else is up to you or extensions.

Hello World

Creating a webpage with less code.

from flask import Flask
app = Flask(__name__)

@app.route("/")
def hello():
    return "Hello World!"

if __name__ == "__main__":
    app.run()
